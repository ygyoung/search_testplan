Testing with Importance

Topics which have more topics linked to it are not showing up higher in search results

  1. Open the HTML5 target |Advanced tab and see that "Include Importance" is selected
  2. Build the target and view the output - search for the word "important"
  3. See that the search results show the topics which start with the words "one important heading with # link" - but the topics with less links show up higher in the search results than the ones that have more links. The topic with no links to it show up first in the list.

Desired

The topics which are linked to from more topics should show up higher in search results than topics which are linked to from less topics.

Desired order:

One important Heading with 6 xref/href links

One important Heading with 5 xref/href links

One important Heading with 4 xref/href links

One important Heading with 3 xref/href links

One important Heading with 2 xref/href links

One important Heading with 1 xref/href links

There are no important headings links
